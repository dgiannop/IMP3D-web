What makes IMP3D different

Box Modeling, refined

Box modeling starts from simple primitives (box, cylinder, plane) and iteratively extrudes, bevels, and connects to form complex shapes. IMP3D streamlines these steps, keeps indices stable, and makes loop & ring operations effortless.

Subdivision Surface

High‑quality Catmull‑Clark subdivision with primary/secondary edge visualization. Edit the coarse mesh and preview the smooth limit surface in real‑time.

UV Unwrapping

Robust UVs with industry‑standard parameterization methods such as LSCM and ABF++, giving predictable, low‑distortion maps for texturing.

Selection that obeys topology

Quad‑aware loops, rings, and reliable picking—optimized for both perspective and orthographic workflows.

Tech Notes

Working with IMP 3D

Start by creating a primitive such as a Box or Quad Sphere. IMP3D supports multiple selection modes available in every tab: Points, Edges, and Polygons.

Most tools work directly on the current selection, or on the entire mesh if nothing is selected. For example, you can select a point, edge, or polygon and move it with the Move Tool. Selecting a polygon and then Shift + Double-Click on an adjacent polygon quickly creates a polygon loop.

With a loop selected, hit the Extrude button and drag to extrude the geometry. At any time, you can increase or decrease the subdivision level using the UI or the - and = keys.

To bake subdivision into the actual mesh, use the Freeze command. For exporting to game engines or other pipelines, use the Triangulate command to convert quads into triangles.

Note: In this free version, some features are currently disabled as development continues.

UV parameterization methods mentioned here (LSCM, ABF++) refer to widely used algorithms for minimizing angular and area distortion when flattening surfaces to 2D for texturing.
